Try it

All endpoints authenticate with Authorization: Bearer <key>. Set it once and reuse it: 

export AOD_API_TOKEN={{ raw_key }}
export AOD_API_URL={{ api_base }}

Health check

curl "$AOD_API_URL/health"

Create an environment

curl -X POST "$AOD_API_URL/environments" \
  -H "Authorization: Bearer $AOD_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "default"}'

Create an agent

curl -X POST "$AOD_API_URL/agents" \
  -H "Authorization: Bearer $AOD_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "hello",
    "runtime": "claude",
    "model": "anthropic/claude-sonnet-4-6"
  }'

Run a session

curl -X POST "$AOD_API_URL/sessions" \
  -H "Authorization: Bearer $AOD_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "<agent-id-from-above>",
    "prompt": "print hello world"
  }'

Head to the dashboard to browse your agents, environments, and sessions.
