Agent on Demand

A REST API for running AI coding agents on sandboxed cloud machines. Define an agent (model + runtime), wire up an environment (packages, env vars, network rules), and start a session. Stream output back over SSE. Multi-turn when you need it.

Agents

A runtime + model pairing. Claude, Codex, Gemini, and more. Optional MCP servers and system prompts.

Environments

Packages, env vars, setup scripts, DNS allow-lists. Encrypted at rest. Versioned.

Sessions

One agent run on one environment. Stream output over SSE. Resume with new prompts.

Powered by Sprites

Every session runs inside a fresh sandbox provisioned on sprites.dev. You bring your own Sprites token and your own model API keys, stored encrypted and scoped per-user. No shared credentials, no credential pooling.

Open source. Contributors welcome.

Agent on Demand is Apache-2.0 and lives on GitHub. We'd especially love help adding more runtimes and model backends. If your favorite agent CLI isn't supported, open a PR, file an issue, or come say hello.
